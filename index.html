Cómo publicarlo en GitHub

Sube esta carpeta a un repositorio y activa GitHub Pages desde la configuración del repositorio. Como es un sitio estático, debería funcionar sin cambios.

Cuando quieras añadir tus imágenes definitivas, reemplaza los archivos dentro de img/chapters/ por imágenes con el mismo nombre.

Sitio generado a partir del material base de la novela. Hecho para una publicación simple, legible y lista para personalizar.
